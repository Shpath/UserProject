Users Application - This application stores the information for users

Dependencies:
    In order to use the application you must have installed the Node.js
        (Link for download: https://nodejs.org/en/download/)

How to start the application ?
    - Open the CMD and go to the directory  of application
    - Write: npm install (In order to install all dependencies)
    - Write: npm start (Starting the server)
    - The application will be served at http://localhost
